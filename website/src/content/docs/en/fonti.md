SkillFishOS is an integration effort: it brings together the work of many open source communities and projects. This page collects the sources cited in the documentation and the projects the system is built on.

## BC-250 hardware documentation

- **[bc250.info](https://bc250.info)** — community wiki about the BC-250 board.
- **[elektricm.github.io/amd-bc250-docs](https://elektricm.github.io/amd-bc250-docs)** — detailed technical documentation.

## BC-250-specific projects

- **[cyan-skillfish-governor (Magnap)](https://github.com/Magnap/cyan-skillfish-governor)** — SMU governor for GPU frequencies.
- **[bc250_smu_oc (bc250-collective)](https://github.com/bc250-collective/bc250_smu_oc)** — overclock/undervolt via the SMU.
- **[bc250-40cu-unlock (duggasco)](https://github.com/duggasco/bc250-40cu-unlock)** — unlocking the 40 Compute Units.
- **[bc250_memcfg (fanoush)](https://github.com/fanoush/bc250_memcfg)** — memory configuration.

## System base

- **[Debian](https://www.debian.org/)** — the base distribution (*sid* branch).
- **[KDE Plasma](https://kde.org/plasma-desktop/)** — the desktop environment.
- **[linux-tkg (Frogging-Family)](https://github.com/Frogging-Family/linux-tkg)** — kernel build recipe.
- **[Mesa / RADV](https://docs.mesa3d.org/drivers/radv.html)** — open source Vulkan graphics drivers.
- **[amdgpu driver](https://docs.kernel.org/gpu/amdgpu/)** — AMD kernel driver.
