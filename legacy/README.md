# Legacy

These scripts are kept **for historical reference only** and are **not used** by current SkillFishOS.

They were the project's earliest, manual setup attempt on an older base (Debian *forky* + **Cinnamon**, an early kernel pulled from external URLs). The project has since moved to **Debian sid + KDE Plasma**, a self‑built `linux-tkg` kernel, and a reproducible live‑build ISO recipe.

For the current, supported way to build and configure the system, see:

- [../README.md](../README.md)
- [../docs/OPTIMIZATIONS.md](../docs/OPTIMIZATIONS.md)
- [../docs/BUILD.md](../docs/BUILD.md)

Do not run these scripts on a current install.
